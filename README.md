# Project Overview

Ultimate White Pagoda is an open-source project to create an extremely detailed, realistic Japanese five-story pagoda using only white paper and a few simple reinforcement materials.

The goal is to design a printable paper craft kit that anyone can build using A4 paper.

---

# Project Goals

- 🏯 Highly realistic five-story pagoda
- 🤍 White-only architectural model
- 📄 Printable on A4 paper
- ✂️ Cut, fold, and glue assembly
- 📐 Precise dimensions
- 📚 Fully illustrated instruction manual
- 🌍 Open-source design

---

# Specifications (Prototype v0.1)

| Item | Value |
|------|------|
| Height | 300 mm |
| Base Size | 180 × 180 mm |
| Color | White |
| Paper | A4 0.20 mm cardstock |
| Reinforcement | Paper, chopstick, drinking straw |
| Estimated Parts | ~150 |

# Materials

- White cardstock (0.20 mm)
- Craft knife
- Metal ruler
- Cutting mat
- White glue
- Wooden chopstick
- Plastic drinking straw

---

# Design Principles

- White only
- No painting required
- Strong internal structure
- Beginner-friendly assembly
- Modular parts
- Printable at 100% scale
