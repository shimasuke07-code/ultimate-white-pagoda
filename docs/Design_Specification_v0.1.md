# Design Specification v0.1

## Project

Ultimate White Pagoda

## Prototype

Version 0.1

## Overall Dimensions

| Item | Size |
|------|------|
| Height | 300 mm |
| Base | 180 × 180 mm |
| Paper | A4 0.20 mm cardstock |

## Materials

- White cardstock
- White glue
- Wooden chopstick
- Drinking straw

## Design Rules

- White only
- Printable on A4
- Modular construction
- Parts numbered sequentially
- 100% printable scale

## Part Categories

A = Base

B = First Floor

C = Second Floor

D = Third Floor

E = Fourth Floor

F = Fifth Floor

G = Finial (Sorin)

## Prototype Goals

- Strong structure
- Easy assembly
- Accurate proportions
- Printable templates
