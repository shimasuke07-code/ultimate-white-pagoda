# Master Grid v1.0

## Units

- All dimensions are in millimeters (mm).

## Global Origin

Origin (0,0,0) is located at the center of the base plate on the ground surface.

```
           Z
           ↑
           │
           │
           ●────→ X
          /
         /
        Y
```

## Coordinate System

| Axis | Direction |
|------|-----------|
| X | Left ↔ Right |
| Y | Front ↔ Back |
| Z | Bottom ↔ Top |

## Base

| Item | Value |
|------|------:|
| Width | 180 |
| Depth | 180 |
| Height | 0 |

## Global Rules

- Symmetrical about X and Y.
- All floors are centered on the origin.
- The central pillar lies on the Z-axis.
- Roof overhang is measured from the floor edge.
